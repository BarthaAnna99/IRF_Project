A beadandó keretein belül egy Kanban táblázatot szeretnék létrehozni.
A fő funkciói az alábbiak lennének:
- grafikus drag and drop felület a Kanban táblázathoz. AZ egyes UserStorykra dublán kattintva szerkezthetjük is ezeket. A feladatok fontosságát tekintve színezhetjük is az elemek hátterét
- Ugyanezen a felületen egy gomra kattintva az adott időszakra szóló táblázatot megnyithatjuk Excelben.
- Lesz egy jelentés menü is, ahol összegezve láthatjük az egyes emberekre vonatkozó feladatokat, időszakokról jelentésket stb.
- Beállítás menüpontban pedig hozzáadhatunk új embereket és időszakokat az alkalmazáshoz
